Bridges Collection

Suspension Bridge

A suspension bridge is a type of bridge where the deck is hung from cables that are suspended between towers.

Cable-stayed Bridge

A cable-stayed bridge is a bridge in which the deck is supported by cables directly attached to towers.

Cantiliver Bridge

A cantilever bridge is a type of bridge built using cantilevers, which are structures that extend horizontally and are supported at only one end. These bridges use massive counterweights and strong beams to balance the structure and allow it to span long distances without the need for central supports.

Arch Bridge

An arch bridge is a type of bridge where the main supporting elements are curved arches, distributing weight outward to the supports at each end, allowing for greater spans without intermediate supports

Beam Bridge

Beam bridge are the simplest type of bridge, consisting of a horizontal beam supported at each end, and are often used for shorter spans. They are also known as stringer bridges

Truss Bridge

A truss bridge is a type of bridge whose load-bearing superstructure is composed of a truss.
